## Dev run
`npm run dev`
If browser didn't open automatically, console will show you address to open

## Production build
`npm run build-prod` - only Unix! \
`npm run build-prod-win` - only Windows

## Dedicated run
`npm run build && npm run server` - run with simple http-server
